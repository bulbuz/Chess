MAIN STRUCTURE OF CHESS PROGRAM
===============================

main loop {
1. print board

2. take input

3. convert input

4. validate user input
    sub steps:
- white or black turn
- get piece type
- validate piece movement

5. move piece

6. check if won
}

CHECK MATE - get the position of the king and see if it is attacked and has no escape squares
